# /lib folder

Place your MySQL JDBC connector JAR file here.

## Download Link
https://dev.mysql.com/downloads/connector/j/

## Steps
1. Go to the link above
2. Select "Platform Independent" from the dropdown
3. Download the ZIP file
4. Extract it
5. Copy `mysql-connector-java-8.x.x.jar` into this `/lib` folder

## Why is the JAR not included?
The MySQL connector JAR is not included in this repository
because it is a third-party library. You must download it
separately from the official MySQL website.
